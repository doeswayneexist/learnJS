Things To Remember

An expression is something that has a value. I.e., 5, "strings", true.

The collection of variables and their values that exist at a given time is called the environment.

Areturnstatement without an expression after it will cause the function to returnundefined.

A variable created inside a function exists only inside that function.

When a function is defined inside another function, its local environment will be based on the local environment that surrounds it instead of the top-level environment.

A function defined inside another function retains access to the environment that existed in that function at the point when it was defined.
